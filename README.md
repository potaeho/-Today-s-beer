# 🍺 오늘의 맥주

> 5가지 맛 프로파일로 내 취향에 딱 맞는 맥주를 찾아보세요

---

## 프로젝트 소개

**오늘의 맥주**는 수제맥주 소비자가 새로운 맥주를 고를 때 겪는 **맛 실패**를 줄이기 위한 맛 프로파일 기반 맥주 탐색 앱입니다.

기존 평점·스타일 정보는 개인 취향 수준의 디테일을 담지 못합니다.  
단순히 "단맛 있음"이 아니라 **파인애플의 단맛인지, 다크초콜릿의 단맛인지** 구분할 수 있어야 진짜 취향 매칭이 가능합니다.

---

## 핵심 기능

### 🎯 5축 맛 프로파일
맥주를 5가지 축으로 수치화해 레이더 차트로 시각화합니다.

| 축 | 설명 |
|---|---|
| 단맛 | 당도 및 당류의 풍미 강도 |
| 신맛 | 산미 및 발효 특유의 신 맛 |
| 쓴맛 | 홉·로스팅에서 오는 쓴 맛 |
| 몰티함 | 맥아에서 오는 빵·곡물 풍미 |
| 부드러움 | 마우스필 및 목넘김 질감 |

### 🏷️ 세분화된 해시태그
각 축마다 구체적인 맛의 뉘앙스를 해시태그로 선택할 수 있습니다.

- **단맛** → 🍍 #파인애플 / 🥭 #망고 / 🍫 #다크초콜릿 / 🍯 #꿀 ...
- **신맛** → 🍋 #레몬 / 🍇 #청포도 / 🥛 #요거트 ...
- **쓴맛** → 🌿 #허브 / 🍊 #시트러스껍질 / ☕ #로스티드 ...
- **몰티함** → 🍞 #토스트 / 🥖 #오븐에구운빵 / 🌾 #곡물 ...
- **부드러움** → 🥂 #크리미 / ✨ #실키 / 🏜️ #드라이 ...

### ⭐ 재구매 의사 별점
맥주를 마신 후 재구매 의사를 5점 만점으로 평가합니다.

## 화면 구성

```
홈 화면
├── 검색 (이름 · 스타일 · 태그)
├── 카테고리 탭 (라거 · 에일 · IPA · 사워 · 스타우트)
├── Today's Pick
└── 카테고리별 맥주 목록

맛 프로파일 입력
├── 레이더 차트 (실시간 반영)
└── 5축 슬라이더 (직접 수치 입력 가능)

해시태그 선택
├── 축별 해시태그 그룹
├── 선택된 태그 하단 고정 표시
└── 재구매 의사 별점

확인 모달 → 결과 페이지
```
